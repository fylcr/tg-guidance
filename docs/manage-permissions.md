# 管理权限

这是管理群组的第一步——知道你能做什么。

拥有管理权限的人员分为两种——一种是所有者，另一种是管理员。一个群组只有一个所有者，但是可以有多个管理员。

我们首先来看一看所有者可以干什么。

## 1. 所有者

如果你创建了这个群组的话，那么你就是这个群组的所有者。

> 不过所有者除了自己创建群组这种途径外，还有转让这种途径，这之后会讲到。

首先我要说的是所有者拥有最高权限，之后我们讲管理员的权限的时候很多操作就会与所有者的操作一样，不会再重复一遍，所以即使你不是所有者的话你也不要跳过这一部分去看管理员的那一部分。

还有一个很重要的点，所有的设置别忘记保存了。

### (3) 能设置群组类型

:::tabs
== Windows

打开`群组类型`子菜单。

![Windows](photo/2-6-1-1.webp)

最上面两个单选框是设置群组类型的，其中：

* `公开群组`：没有加入群组的人也能看到群组内的消息。

* `私密群组`：只有加入群组的人才能看到群组内的消息。

如果你把群组设置成`公开群组`，那么你需要给群组设置一个用户名（或者说链接），只需要在`链接`输入你喜欢的用户名就行了

>
>![Windows](photo/2-6-1-4.webp)

>![Windows](photo/2-6-1-5.webp)

![Windows](photo/2-6-1-6.webp)

这两个群组类型都有的`内容保护`则是限制群组里的用户保存、复制和转发这条群组内的消息的。

### (6) 能管理用户权限

这个是最重要的功能，我们管理群组主要使用的就是这个功能。

:::tabs
== Windows

打开`权限管理`子`菜单。

![Windows](photo/2-11-1-1.webp)

我们从上到下一个一个的讲解：

首先是全体成员权限，这里修改的内容是对全体成员起效的。

1. `发送消息`：用户能不能在群里发送纯文字消息。这个选项关闭后，用户仍可以发送除了嵌入链接预览这一类的其余八类媒体文件。
2. `发送媒体文件`：这有九个子项，点开后面的折叠框我们可以看到这具体的九项：
  ![Windows](photo/2-11-1-2.webp)
    * `照片`：取消勾选后用户不能发送照片。
        >要注意的一点是 Telegram 有两种发送图片的方法。一种是压缩后在发送的，就是这里说的照片。
        >
        >![Windows](photo/2-11-1-3.webp)
        >
        >发出来的消息是这样的：
        >
        >![universe](photo/2-11-0-1.webp)
        >
        >另一种是不压缩以文件形式发送的。
        >
        >![Windows](photo/2-11-1-4.webp)
        >
        >发出来的消息是这样的：
        >
        >![universe](photo/2-11-0-2.webp)
        >
        >这里只限制第一种发送照片的方法，而不限制第二种方法
    * `视频文件`：取消勾选后用户不能发送视频文件。
    * `视频消息`：取消勾选后用户不能发送视频消息。
        >什么是视频消息呢？我们在消息输入框为空的时候最右侧会有一个`📸`按钮。
        >
        >![Windows](photo/2-11-1-5.webp)
        >
        >长按以后会录制一段不超过60s的圆形视频。和视频文件一样，这个视频在别人看到时会自动静音播放。
        >>如果处于语音消息模式，左键单击一次后就会变成视频消息模式。
    * `音乐`：取消勾选后用户不能发送音乐。
    * `语音消息`：取消勾选后用户不能发送语音消息。
        >什么是语音消息呢？我们在消息输入框为空的时候最右侧会有一个`🎙`按钮。
        >
        >![Windows](photo/2-11-1-6.webp)
        >
        >长按以后会录制一段音频，就像微信那样。
        >>如果处于视频消息模式，左键单击一次后就会变成语音消息模式。
    * `文件`：取消勾选后用户不能发送文件。
        >没有压缩的照片也会被当做文件，但是视频文件不会被禁止。
    * `贴纸或 GIF 动态图`：取消勾选后用户不能发送贴纸或 GIF 动态图。
    * `嵌入链接预览`：取消勾选后用户发送消息里的链接不能有预览。
        >我们在输入框里输入了一个链接，Telegram 会自动生成一个预览，这个就是嵌入链接预览。
        >
        >![Windows](photo/2-11-1-7.webp)
        >
        >发出去的消息是这样的：
        >
        >![universe](photo/2-11-0-3.webp)
        >
        >但是当 Telegram 自动生成预览后改变输入框内的消息并不会导致嵌入链接预览发生改变,这将会导致嵌入链接预览与文字内容是脱节的，而我们的自动反垃圾系统很多只根据文字内容判断,这就给打广告提供很大的便利。不过嵌入链接预览是一个很方便的功能，关闭以后会带来一定的不便，之后我们会详细讲到这一点。
    * `发起投票`：取消勾选后用户不能发起投票。
3. `添加成员`：用户能不能把其他人拉进这个群里。
4. `创建话题`：用户能不能创建新话题，我建议把这个关掉。
    >不过用户创建的话题可以被我们或者他们自己（创建者）删掉。
5. `置顶消息`：用户可不可以置顶消息，但是公开群组是不能打开的。
    >当你右键一个消息时，可以在弹出的菜单里找到置顶信息这一选项，之后会弹出提示并选择是否通知所有成员，如果勾选了那么在置顶时会通知所有人，不勾选其他人不会收到通知，置顶后则在群组最上方有提示。
6. `修改群组信息`：用户可不可以修改群组信息，但是公开群组是不能打开的。
    >这里的群组信息包括群组名称、头像和简介。

这六项我们限制任意一项后，会多出一个`不限制助推者`，这个是默认关闭的，通常也不会开启。
>什么是助推？你可以理解成拼多多砍一刀的好友助力，积累足够多的助推次数，群组可以使用更高级的功能。不过组推只能由高级用户助推，所以一般情况下助推都是可有可无的东西。~~（因为高级用户没有那么多。）~~

接着是`为消息收取星星`，开启后用户支付星星后才能发送消息。

`慢速模式`限制用户每两次消息之间的时间间隔，比如10s就是发送一条消息10s后才能发送下一条消息。

`例外`就是被特殊对待的用户，在这里可以禁言用户，或者个用户特殊的权限。如果想让这个用户再和大家一样，只需要删除就行了。

`被移除的用户`相当于被封禁，除非所有者或管理员解除限制或手动添加，这里的用户永远都不能重新加入到群里。

## 2. 管理员

管理员所能干的事情和所有者一样，所以很多的细节都不会再说一遍。

这里只强调一个点，知道管理员的自己的权限很重要。即使在设置管理员的时候没有给任何权限，但是这个用户还是管理员，他/她/它还是能：

* 查看近期操作
* 查看管理员列表（尤其是匿名发送的时候）

而且由于管理员头衔的存在，即使没有管理群组的目的，一名用户有可能仅仅因为娱乐目的被授予管理员。但请记住，这个看似什么用都没有的头衔也会使用户从普通用户跻身进入管理员的行列。
>这也是很多群管理机器人有权限检测（检测一名管理员是否拥有某项管理权限）这一设定，否则可能仅仅一个空有其名的头衔就让用户获取了设置机器人的权限。但是权限检测这一设定导致一名管理员权限不够大的时候不能设置机器人从而影响管理效率。所以在设置新管理员一定要权衡好这个管理员的权限，不能太大导致群组存在风险，也不能太小导致很多事情无法实现。

这一节我们学习了管理员基础知识与技能，下一节将会讲解常见的不和谐分子类型与相应的应对技巧。
