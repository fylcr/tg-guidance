# 违禁操作

当机器人根据我们的设置自动识别到违规行为时，需要自动进行操作，这就是违禁操作。不过这些操作也是可以手动执行的，你可以直接使用命令来执行这些操作。

## 踢出、禁言与封禁

### 1. 踢出

对于想要踢出的人，直接使用命令`/kick @用户名`，或者回复这个人的违规消息命令`/kick`。这样的话 MissRose 会自动把用户踢出群组，但是用户发出的消息不会被删除，且用户可以随时再次加入群组。

如果想要自动操作的话，操作模式直接填`kick`就行了。

### 2. 禁言

对于想要踢出的人，直接使用命令`/mute @用户名`，或者回复这个人的违规消息命令`/mute`。这样的话 MissRose 会把这个人禁言，用户仍然在群里，能看到群里其他人发的消息，也可以使用表情回应其他人的消息。
