# MissRose

MissRose 可以说是目前最受欢迎的群管理机器人之一，比较熟悉的人会称之为大妈~~（虽然名字为小姐）~~。

打开 [MissRose](https://t.me/MissRose_bot) 的资料页，将机器人添加进群组并给足权限。
>其实默认的权限就可以了，如果真的不想给太多权限的话，至少给：
>
>* 删除消息
>* 封禁用户
>虽然有些功能不能使用，但只有这些权限已经不妨碍 MissRose 管理群组了。

MissRose 可能是唯一一个有[专门文档网站](https://missrose.org/)的群管理机器人，可以打开参考一下。

添加到群组后，你需要在群组里输入`/start@MissRose_bot`命令。之后 MissRose 会回复一条消息。

![Missrose](../photo/bot-1-1.webp)

点击下面的按钮进行人机验证。
>这是为了防止 MissRose 的服务器被 spam 乱搞而炸掉的。~~（这可是血的教训）~~

发送`/setlang`，将 MissRose 的语言设置成你想要的语言。（例如中文或繁体中文）
