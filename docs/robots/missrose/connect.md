# 将 MissRose 连接到私聊

首先谈一谈为什么将 MissRose 连接到私聊？

如果直接在群里设置机器人的话，所有的设置都会暴漏给群聊成员。即使你删除了命令，但是有些第三方客户端会有防删除机制，你的命令删除得再快也会被看到，从而导致被 spam 针对。连接到私聊后就可以直接在私聊里发送命令来设施机器人了，避免命令泄露给其他用户。

下面是几种方法来连接到群组：

第一种是我常用的方法。

在群里发送`/connect@missrose_bot`,之后 MissRose 会恢复一条消息。

![Missrose](../photo/bot-1-2-1.webp)

点击下面的按钮以跳转到私聊，这样的话就连接到私聊了。

>不过这样 connect 命令也会暴漏给其他用户。

第二种是只依赖于私聊的方法。

在私聊里发送`/connect @群用户名`（群用户名就是公开群组设置链接的时候你所输入的名字，这之前说过）

这样也能连接到私聊。

如果群组为私密群组的话，那么可以使用最麻烦的一种。

首先在群组里发送`/id`，MissRose 会回复群组 id。

![Missrose](../photo/bot-1-2-2.webp)

>群组 id 一定是负数。

之后在私聊里回复`/connect 群组id`，这样就成功连接到私聊了

如果你之前连接到了多个群组，那么你在私聊里发送`/connect`就可以快速地切换连接的群组。

![Missrose](../photo/bot-1-2-3.webp)

>MissRose 最多只会保存5次连接记录，或者说可以同时连接5个群组。

到这里我们已经成功连接到私聊了，完成了基本的设置。
